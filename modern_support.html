Frequently Asked Questions

How do I add Guard-shin to my server?
You can add Guard-shin to your server by clicking the "Add to Discord" button in the sidebar or by visiting our homepage. You'll need to have the "Manage Server" permission in the Discord server you want to add the bot to.
What permissions does Guard-shin need?
Guard-shin requires several permissions to function properly, including "Manage Messages", "Kick Members", "Ban Members", "Manage Roles", and more. When you add the bot to your server, you'll see a list of all required permissions. For optimal performance, we recommend using the default permission setup.
How do I set up auto-moderation?
You can set up auto-moderation using the g!automod command. This will allow you to configure filters for inappropriate content, spam protection, and more. Use g!automod help for a detailed guide on all available options.
What's the difference between free and premium?
The free version of Guard-shin includes basic moderation commands, auto-moderation for text content, server statistics, and a limited music player. Premium unlocks advanced features like raid protection, custom welcome images, unlimited music player, custom auto-responders, advanced analytics, and priority support. Check out our Premium page for a detailed comparison.
How do I change the bot's prefix?
You can change the bot's prefix using the g!prefix command. For example, to change the prefix to "!", use g!prefix !. This feature is available to premium users only.
How does the music player work?
The music player allows you to play music in voice channels. Use g!play [song name or URL] to add a song to the queue. You can manage the queue with commands like g!skip, g!pause, g!resume, and g!queue. Free users can play up to 5 songs per server, while premium users have unlimited access.
