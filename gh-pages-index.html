Available Commands

Here are some of the most commonly used commands:

Command Description Usage

!kick Kick a user from the server !kick @user [reason]
!mute Mute a user temporarily !mute @user [duration] [reason]
!warn Issue a warning to a user !warn @user [reason]
!clear Clear messages from a channel !clear [amount]
