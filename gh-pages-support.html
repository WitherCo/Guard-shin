Frequently Asked Questions

How do I add Guard-shin to my server?

You can add Guard-shin to your Discord server by clicking the "Add to Discord" button in the sidebar. This will redirect you to Discord's authorization page where you can select your server and grant the necessary permissions.

What permissions does Guard-shin need?

Guard-shin requires certain permissions to function properly, including:

  • Read Messages & View Channels - to see and respond to commands
  • Send Messages - to respond to commands and send notifications
  • Manage Messages - for moderation features
  • Kick Members & Ban Members - for moderation actions
  • Manage Roles - for auto-role and moderation features

For full functionality, Administrator permission is recommended, but you can customize permissions based on your needs.

How do I change the bot's prefix?

You can change Guard-shin's command prefix using the !prefix command. For example, to change the prefix to ?, type !prefix ? in any channel where the bot can see your messages.

Alternatively, you can change the prefix in the Settings page on this dashboard.

What's the difference between the free and premium versions?

The free version of Guard-shin includes essential moderation and security features, while Premium adds:

  • Music commands
  • Custom welcome images
  • Advanced auto-moderation
  • Auto-role assignment
  • Advanced analytics
  • Priority support

Check out our Premium page for more details and subscription options.

How do I report a bug or suggest a feature?

You can report bugs or suggest features through our GitHub repository, by joining our Discord support server, or by using the contact form on this page.

When reporting bugs, please provide as much detail as possible, including steps to reproduce the issue, expected behavior, and screenshots if applicable.
