Getting Started

Guard-shin is an advanced Discord moderation and security bot designed to protect your server against unwanted behavior, raids, spam, and other disruptive activities. This documentation will help you get started with configuring and using Guard-shin to its full potential.

Inviting the Bot

To add Guard-shin to your server, click the "Add to Server" button on our homepage or click here. You need "Manage Server" permissions to add a bot to your server.

We recommend giving Guard-shin Administrator permissions for full functionality. If you prefer not to, you'll need to make sure it has at minimum these permissions: Manage Server, Manage Roles, Manage Channels, Kick Members, Ban Members, Manage Messages, View Channels, Send Messages, Embed Links, Attach Files, Read Message History, and Add Reactions.

Initial Setup

Once Guard-shin is on your server, you can start using it immediately with default settings. Use the following commands to get started:

/help - View all available commands /automod setup - Start the auto-moderation configuration process /verify setup - Configure the verification system /logs setup - Set up logging channels /welcome setup - Configure welcome messages

Moderation System

Guard-shin offers a comprehensive set of moderation tools to help server administrators and moderators manage their communities effectively.

Basic Moderation Commands

Here are the essential moderation commands available to server moderators:

Command Description Required Permission
/ban Ban a user from the server Ban Members
/kick Kick a user from the server Kick Members
/mute Temporarily mute a user Moderate Members
/warn Issue a warning to a user Moderate Members
/purge Bulk delete messages Manage Messages

Infraction Management

Guard-shin keeps a record of all moderation actions taken against users. You can view and manage these infractions using the following commands:

/warnings [user] - View warnings for a specific user /infractions [user] - View all infractions for a user /clearwarnings [user] - Clear all warnings for a user

Timed Punishments

You can set temporary punishments that automatically expire after a specified duration:

/mute [user] [duration] [reason] - Mute for a specific time (e.g., 1h, 3d) /ban [user] [duration] [reason] - Temporary ban /timeout [user] [duration] [reason] - Discord timeout feature

Auto Moderation

Guard-shin's auto-moderation system automatically detects and takes action against various forms of undesirable content and behavior without requiring manual intervention.

Setting Up Auto-Moderation

To get started with auto-moderation, use the following command:

/automod setup

This will guide you through the process of setting up various auto-moderation features. You can also configure individual aspects:

Content Filtering

Guard-shin can automatically detect and remove unwanted content:

/automod filter add [word/phrase] - Add words or phrases to filter /automod filter remove [word/phrase] - Remove words from filter /automod filter list - View current filter list

Anti-Spam Protection

Configure protection against message spam:

/automod antispam enable/disable - Toggle spam protection /automod antispam threshold [number] - Set spam detection threshold /automod antispam action [warn/mute/kick] - Set action for spam detection

Link Filtering

Control which links are allowed in your server:

/automod links allow/deny - Toggle link filtering /automod links whitelist add [domain] - Add domains to whitelist /automod links whitelist remove [domain] - Remove domains from whitelist

Raid Protection

Protect your server from coordinated join attacks:

/automod antiraid enable/disable - Toggle raid protection /automod antiraid threshold [number] - Set raid detection threshold /automod antiraid action [verify/quarantine/kick] - Set action for raid detection
Raid protection may require additional setup with the verification system for maximum effectiveness. See the Verification System section for more details.

Verification System

Guard-shin offers a verification system to ensure that only legitimate users can access your server. This is particularly useful for protecting against raids and bot accounts.

Basic Setup

To set up the verification system:

/verify setup

This will guide you through the process of setting up:

  • A verification channel where new members will see instructions
  • A verification role that is assigned after successful verification
  • Verification message customization

Verification Methods

Guard-shin supports several verification methods:

Method Description
Button Verification Users click a button to verify (simplest)
Captcha Verification Users must solve a captcha challenge
Question Verification Users must answer a custom question

To configure the verification method:

/verify method [button/captcha/question] /verify captcha enable/disable /verify question set [question] [answer]

Quarantine Settings

During raids, Guard-shin can place suspicious users in quarantine:

/verify quarantine role [role] - Set the quarantine role /verify quarantine channel [channel] - Set the quarantine channel

Music System

Guard-shin includes a high-quality music player with support for various sources including YouTube, Spotify, and SoundCloud.

Playing Music

To play music, use the following commands:

/play [song name or URL] - Play a song or add to queue /search [query] - Search for songs and select from results

Guard-shin supports various sources:

  • YouTube videos and playlists
  • Spotify tracks, albums, and playlists
  • SoundCloud tracks and playlists
  • Direct MP3 links

Player Controls

Control the music player with these commands:

/pause - Pause the current song /resume - Resume playback /skip - Skip to the next song /previous - Play the previous song /stop - Stop playback and clear the queue /seek [time] - Seek to a specific position in the song /loop [track/queue/off] - Set loop mode /shuffle - Shuffle the queue /volume [1-100] - Adjust player volume

Queue Management

Manage the song queue with these commands:

/queue - View the current queue /remove [position] - Remove a song from the queue /clear - Clear the entire queue /jump [position] - Jump to a specific song in the queue

Frequently Asked Questions

General Questions

Q: How do I change the bot's prefix?

A: Use the command /prefix set [new prefix] to change the prefix. For example, /prefix set ! would change it to !.

Q: Can I use Guard-shin alongside other moderation bots?

A: Yes, Guard-shin is designed to work alongside other bots. Just make sure role hierarchies are properly set up to avoid conflicts.

Q: How can I report bugs or suggest features?

A: Join our Support Server to report bugs or suggest features.

Troubleshooting

Q: The bot isn't responding to commands.

A: Check the following:

  • Ensure the bot has the necessary permissions in the channel
  • Make sure you're using the correct prefix or slash commands
  • Check if the bot is online in your server's member list

Q: Auto-moderation isn't working.

A: Verify that:

  • Auto-moderation is enabled with /automod setup
  • The bot has permission to manage messages and moderate members
  • The bot's role is higher in the hierarchy than the members it needs to moderate

Q: How do I exempt certain channels or roles from auto-moderation?

A: Use the following commands:

/automod ignore channel [channel] - Ignore a specific channel /automod ignore role [role] - Ignore users with a specific role
