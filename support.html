Frequently Asked Questions

How do I add Guard-shin to my server?

Adding Guard-shin to your server is simple. Click the "Add to Discord" button on our website, authorize the bot with your Discord account, and select the server where you want to add it. Make sure you have the "Manage Server" permission on that server.

What permissions does Guard-shin need?

Guard-shin requires the "Administrator" permission for full functionality. However, if you prefer to limit its permissions, the bot needs at minimum: Manage Roles, Manage Channels, Kick Members, Ban Members, Manage Messages, View Audit Log, and Send Messages. Some features may not work without the proper permissions.

How do I set up custom welcome messages?

Use the !welcome command to set up custom welcome messages. For example: !welcome set #welcome-channel Welcome {user} to our server! will send a welcome message when new members join. You can use variables like {user}, {server}, {count}, etc. to personalize your messages.

My premium subscription isn't working. What should I do?

If your premium subscription isn't working properly, try the following steps:
1. Make sure your payment was successful and not declined.
2. Use the !premium status command to check your subscription status.
3. Try refreshing your connection with !premium refresh.
4. If the issue persists, please contact our support team with your transaction ID ready.

Can I customize the bot's prefix?

Yes, you can customize the prefix using the !prefix command. For example, !prefix $ will change the prefix from ! to $. After changing the prefix, you'll need to use the new prefix for all commands (e.g., $help instead of !help).
