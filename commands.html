Command Description Usage
!ban
Ban a user from the server with an optional reason.
Options: user (User), reason (String, optional)
 !ban @user [reason]
!unban
Unban a user from the server.
Options: user (User ID)
 !unban 123456789012345678
!kick
Kick a user from the server with an optional reason.
Options: user (User), reason (String, optional)
 !kick @user [reason]
!mute
Mute a user, preventing them from sending messages.
Options: user (User), duration (String, optional), reason (String, optional)
 !mute @user 1h spamming
!unmute
Unmute a previously muted user.
Options: user (User)
 !unmute @user
!warn
Issue a warning to a user and log it.
Options: user (User), reason (String, optional)
 !warn @user inappropriate language
!autorole Premium
Automatically assign roles to new members.
Options: status (String), role (Role)
 !autorole enable @Member
!lockdown
Lock down a channel or the entire server, preventing members from sending messages.
Options: channel (Channel, optional), reason (String, optional)
 !lockdown #general raid in progress
