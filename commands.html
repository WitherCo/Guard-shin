Bot Commands

Server Moderation

ban

Ban a user from the server with an optional reason.

/ban user:[user] reason:[optional reason]

kick

Kick a user from the server with an optional reason.

/kick user:[user] reason:[optional reason]

mute

Temporarily mute a user in the server.

/mute user:[user] duration:[time] reason:[optional reason]

warn

Warn a user and add to their warning count.

/warn user:[user] reason:[reason]

purge

Delete a specific number of messages from a channel.

/purge amount:[number] user:[optional user]

unban

Unban a user from the server.

/unban user:[user ID or username]

warnings

View warnings for a specific user.

/warnings user:[user]

timeout

Timeout a user for a specified duration.

/timeout user:[user] duration:[time] reason:[optional reason]

Auto Moderation Settings

automod setup

Setup automatic moderation for your server.

/automod setup

automod filter

Configure word and phrase filters for your server.

/automod filter add:[word/phrase]

automod antispam

Configure anti-spam settings for your server.

/automod antispam [enable/disable] threshold:[number]

automod links

Configure link filtering for your server.

/automod links [allow/deny] whitelist:[domains]

automod antiraid

Configure anti-raid protection settings.

/automod antiraid threshold:[number] action:[action]

automod logs

Set a channel for auto-moderation logs.

/automod logs channel:[channel]

Verification System

verify setup

Set up the verification system for your server.

/verify setup role:[role] channel:[channel]

verify captcha

Enable or disable captcha verification.

/verify captcha [enable/disable]

verify message

Set the verification message for new members.

/verify message [text]

verify logs

Set a channel for verification logs.

/verify logs channel:[channel]

Music System

play

Play a song or playlist from YouTube, Spotify, or SoundCloud.

/play query:[song name or URL]

pause

Pause the currently playing song.

/pause

resume

Resume the paused song.

/resume

stop

Stop the player and clear the queue.

/stop

skip

Skip the current song.

/skip

queue

View the current song queue.

/queue

shuffle

Shuffle the current song queue.

/shuffle

volume

Change the player volume.

/volume level:[1-100]

Utility Commands

help

Display available commands and how to use them.

/help command:[optional specific command]

ping

Check the bot's response time and latency.

/ping

userinfo

Display information about a user.

/userinfo user:[user]

serverinfo

Display information about the server.

/serverinfo

avatar

Get a user's avatar.

/avatar user:[user]

stats

Display bot statistics.

/stats

Administrative Commands

lockdown

Lock a channel or the entire server.

/lockdown channel:[channel] reason:[optional reason]

unlock

Unlock a previously locked channel.

/unlock channel:[channel]

slowmode

Set slowmode for a channel.

/slowmode channel:[channel] seconds:[time in seconds]

role

Add or remove a role from a user.

/role add/remove user:[user] role:[role]

massrole

Add or remove a role from multiple users.

/massrole add/remove role:[role] [filter options]

Configuration Commands

Change the bot's prefix for this server.

/prefix set:[new prefix]

welcome

/welcome channel:[channel] message:[message]

goodbye

Set up goodbye messages for leaving members.

/goodbye channel:[channel] message:[message]

permissions

Configure command permissions for roles.

/permissions command:[command] role:[role] allow/deny
