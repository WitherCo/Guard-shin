Commands

Browse and learn how to use Guard-shin's commands. Some commands are only available with a premium subscription.

Moderation Commands

ban

Moderation

Ban a user from the server with an optional reason.

g!ban @user [reason]
Requires: Ban Members

kick

Moderation

Kick a user from the server with an optional reason.

g!kick @user [reason]
Requires: Kick Members

mute

Moderation

Mute a user for a specified duration, preventing them from sending messages.

g!mute @user [duration] [reason]
Requires: Manage Roles

purge

Moderation

Delete a specified number of messages from a channel.

g!purge [amount]
Requires: Manage Messages

warn

Moderation

Warn a user for rule violations with an optional reason.

g!warn @user [reason]
Requires: Manage Messages

lockdown

Moderation

Lock a channel during raids or emergencies, preventing new messages.

g!lockdown [duration]
Requires: Manage Channels
Premium

Utility Commands

help

Utility

Display a list of available commands or information about a specific command.

g!help [command]
No special permissions required

info

Utility

Display information about the bot, including uptime and version.

g!info
No special permissions required

serverinfo

Utility

Display detailed information about the server.

g!serverinfo
No special permissions required

userinfo

Utility

Display detailed information about a user.

g!userinfo [@user]
No special permissions required

Music Commands

Unlimited

Music

Setup Commands

setup

Setup

Launch the interactive setup wizard for the bot.

g!setup
Requires: Administrator

prefix

Setup

Change the bot's command prefix for your server.

g!prefix [new prefix]
Requires: Administrator
Premium

autorole

Setup

Set up automatic role assignment for new members.

g!autorole [role]
Requires: Manage Roles
Premium

logs

Setup

Requires: Administrator
